# Exercícios de NumPy

Este repositório reúne cinco notebooks (`.ipynb`) com pequenos exercícios práticos
utilizando a biblioteca **NumPy**. Cada notebook contém código e comentários que
explicam o que está sendo feito.

## Estrutura

Os arquivos disponíveis são:

- [Exercício 1](exercicio1.ipynb) – criação de um array unidimensional e alteração de valores.
- [Exercício 2](exercicio2.ipynb) – criação de um array 3×2, consulta de `shape` e de uma linha.
- [Exercício 3](exercicio3.ipynb) – array 3×2 e seleção de uma coluna.
- [Exercício 4](exercicio4.ipynb) – array 4×5 gerado com `arange` e leitura de uma linha.
- [Exercício 5](exercicio5.ipynb) – fatia de um array 4×5 selecionando linhas e colunas específicas.

Clique nos links acima para abrir cada notebook no VS Code ou em outro ambiente que
suporte Jupyter.

## Requisitos

- Python 3.x
- [NumPy](https://numpy.org/) instalado (`pip install numpy` ou usando um
ambiente virtual como `.venv`).

## Uso

1. Abra o projeto no VS Code ou Jupyter Lab.
2. Abra qualquer um dos arquivos `exercicio*.ipynb` clicando nos links acima.
3. Execute as células para visualizar os resultados e editar se desejar.

Sinta‑se à vontade para adicionar novos exercícios ou documentar resultados
matemáticos nos próprios notebooks.
